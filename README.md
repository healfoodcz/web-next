# healfood.cz

A web application written for HEALFOOD CZ s.r.o. company.

> TODO: web app preview (mini gallery?)

## Software prerequisites

- Docker

or

- Node.js
- NPM

## Start (production)

- If using Docker, then `./cli/up`

otherwise

- `npm install`
- `npm run start:prod`

## Develop

- `npm install`
- `npm run start:dev`
