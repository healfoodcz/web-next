# healfood.cz

A web application written for HEALFOOD CZ s.r.o. company.

> Here should be image previews of the web application

## Software prerequisites

- Docker
- Node.js
- NPM

## Start

- If using Docker, then `./cli/up-prod`

otherwise

- `npm install`
- `npm run start:prod`

## Develop

- If using Docker, then `./cli/up-dev`

otherwise

- `npm install`
- `npm run start:dev`
